Required Composer Packages for 2FA:

1. pragmarx/google2fa-laravel (Google 2FA implementation)
   composer require pragmarx/google2fa-laravel

2. bacon/bacon-qr-code (QR Code generation)
   composer require bacon/bacon-qr-code

Installation command:
cd /opt/webserver/admin-panel && composer require pragmarx/google2fa-laravel bacon/bacon-qr-code
